## 📖 Sobre

Jogo interativo onde o jogador deve **adivinhar um número secreto** gerado aleatoriamente entre **1 e 100**. A cada tentativa, dicas visuais e por voz indicam se o palpite foi **maior**, **menor** ou **igual** ao número sorteado.

Projeto desenvolvido durante os cursos de **Lógica de Programação** da [Alura](https://www.alura.com.br), explorando conceitos fundamentais como manipulação do DOM, eventos e geração de números aleatórios.

---

## ✨ Funcionalidades

- 🎲 **Número aleatório** – Sorteio único a cada partida, sem repetições
- 🎙️ **Feedback por voz** – O jogo fala as dicas usando `responsiveVoice`
- 💡 **Dicas dinâmicas** – "Maior" ⬆️ ou "Menor" ⬇️
- 🔢 **Contador de tentativas** – Quantos palpites até acertar
- 🛡️ **Validação de entrada** – Apenas números entre 1 e 100
- 🔄 **Reinício rápido** – Botão "Novo jogo" sem recarregar a página
- 📱 **Design responsivo** – Funciona em celulares e tablets

## 🎮 Como Jogar

1. Abra o `index.html` no seu navegador
2. Digite um número entre **1** e **100**
3. Clique em **"Chutar"**
4. Siga as dicas até acertar o número secreto
5. Clique em **"Novo jogo"** para jogar novamente

---

## ⚙️ Como Executar

```bash
# Clone o repositório
git clone https://github.com/Kyuxih/numero-secreto.git

# Acesse a pasta
cd numero-secreto

# Abra no navegador
start index.html
```
